##项目介绍：
>&#160; &#160; &#160; &#160;“云南特色商城”采用S2SH框架开发、Mahout做商品推荐（基于用户的协同过滤算法）。
##实现功能：
>* 商品查询
>* 会员登录、用户个人信息管理
>* 购物车
>* 在线支付
>* 邮件、短信服务
>* 后台管理
>* 后台用户权限控制
>* 商品销售报表
>* 基于用户的商品推荐

##所用技术：
>* 显示层：Struts2.3
>* 分层解耦：Spring3.2
>* 持久层：Hibernate3.6
>* 基于用户的商品推荐：Mahout
>* 数据库：MySQL5.5
>* JDK：1.6
>* 单元测试：Junit4.0
>* 版本管理：Git
>* Web容器：Tomcat6.0
>* 操作系统：Windows7
>* 开发工具：MyEclipse8.5
>* 建模工具：IBM.Rational7.0
